📘 Resumo do Curso de HTML

Durante o curso, aprendi tanto o HTML antigo quanto o HTML5, entendendo a evolução da linguagem e as melhorias trazidas na versão mais atual.

Aprendi a montar a estrutura básica de uma página, utilizando elementos como <html>, <head> e <body>, que organizam o conteúdo do site.

Também conheci diversas tags importantes, como títulos, parágrafos, links, imagens e listas, que ajudam a estruturar e exibir as informações.

Além disso, estudei o uso de atributos nas tags, que permitem adicionar funcionalidades, como links e imagens.

Aprendi sobre formulários, utilizados para entrada de dados dos usuários, e sobre as tags semânticas do HTML5, que deixam o código mais organizado e moderno.

Por fim, entendi que o HTML é a base de qualquer site e que o HTML5 trouxe melhorias importantes para a criação de páginas mais completas e interativas.
